Process

Every good product should start with the planning and research step.

Planning and research can help to figure out the trend in the current market and strategies suitable for you.

Follow the trend and strategies, creative ideas will be generated.

In the design and development phase, all creative ideas will be gathered and materialized.

In the testing phase, detailed testing can make the product more perfect. The UI/UX can affect by details so we do care all details on our works.

In the launch phase, all testing should be done and work perfectly. The product will be upload and launch.

1
Planning
The planning phase is the first phase of the project, we will grab all your ideas and research first. You can tell us everything you want on the website, we will suggest the most suitable plan for you based on your ideas.
2
Design
The website layout and architecture will be done in this phase. You can also join the design work with our designer team! Tell us what you want on the layout as we want to give you your most satisfying design.
3
Development
In this development phase, our programmers will follow the UI/UX design, sitemap, and other information to build the website. You can test the functions and layout a few times in this phase.
4
Testing
In this phase, you will have a User Acceptance Testing - UAT, they can test the functions and performance detailedly before launch. After the testing and minor changes, the project can to the final phase.
5
Launch
This is the final phase of the project. The website will be uploaded to the production server and ready to launch. SEO and any digital marketing action can do on the website after domain mapping and launch.

Capsubeans Offical Website

Capsubeans create in Hong Kong in 2015. Capsubeans is designed to be the most carefree and the happiest creature in the universe. In the Capsubeans website, we use some technologies on it, like PixiJS, Materialize CSS, NodeJS, etc. PixiJS is an HTML5 creation engine, it can use for 2D WebGL renderer, and we used it to render characters animation.
Featured Technologies
NodeJS, PixiJS, Materialize CSS

Capsubeans Online Store

Capsubeans Online Store is using Shopify as its e-commerce system, Shopify is an e-commerce website builder. Reduce time and cost is the main reason that uses Shopify, as it has a set of pre-built functions, like global payment and checkout functions. We can focus on layout customization and use less time to develop the website.
Featured Technologies
Shopify, Customize theme

Limzy Kitchen

Limzy Kitchen is a food tutorial website. You can learn different recipes on it, it has descriptions and photos in every step, you can learn the recipes step by step, just like WikiHow. You can also be a tutor on Limzy, upload your recipe and photos. In the Limzy website, we use NodeJS and ExpressJS for the functions technologies, and use Materialize CSS and SASS for the layout development.
Featured Technologies
NodeJS, ExpressJS, Materialize CSS, SASS

Chinaimporter

Chinaimporter is a consultant that helps to import and sell products in China. They help foreign companies to set up a sales channel in China. In this website, we used NodeJS, ExpressJS, and ES6 Javascript, and use SASS on the layout. On the layout technology, we also use AMP on layout programming. AMP can increase the loading speed and user experience, and also can do better SEO.
Featured Technologies
NodeJS, ExpressJS, ES6, SASS, AMP

YCCECE

YCCECE is an education school. They provide different courses in early childhood education. In this website, we used Wordpress as its CMS. Wordpress is an opensource CMS framework, it mainly uses PHP and MySQL as the programming languages. Wordpress can easy to install and switch plugins and themes, and we use custom plugin and theme on this website so we can reduce the time on CMS development time.
Featured Technologies
Wordpress, Customize Plugin, Customize Theme
